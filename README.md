# Ankush Kumar - Developer Portfolio

![Portfolio Preview](https://via.placeholder.com/1200x600.png?text=Ankush+Kumar+Portfolio+Preview) <!-- Feel free to replace this link with an actual screenshot of your portfolio later! -->

Welcome to my personal developer portfolio! This project is designed to showcase my skills, featured projects, and my professional journey as a software developer. It is built with a focus on modern design principles, interactivity, and high performance.

## 🚀 Features

- **Modern UI/UX:** A sleek, dark-themed design with vibrant accents and glassmorphism elements.
- **Smooth Animations:** Integrated framer-motion for fluid page transitions, scroll effects, and hover states.
- **Responsive Design:** Fully responsive and optimized for desktops, tablets, and mobile devices.
- **Dynamic Projects Showcase:** Easily updatable grid displaying my latest work including the SLAY Dating Assistant, ConvertX contributions, and AI-Based Institutional Evaluation System.
- **Interactive Skills Section:** Visual representation of my technical stack and core competencies.

## 🛠️ Built With

This project was built using modern web technologies:

- **[React 18](https://reactjs.org/)** - Core frontend library
- **[Vite](https://vitejs.dev/)** - Next-generation frontend tooling for fast builds
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework for rapid styling
- **[Framer Motion](https://www.framer.com/motion/)** - Production-ready animation library for React
- **[Lucide React](https://lucide.dev/)** - Beautiful and consistent icon toolkit

## 💻 Getting Started (Local Development)

If you'd like to clone this repository and run it locally, follow these steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ankushkumar145/<YOUR-REPO-NAME>.git
   cd <YOUR-REPO-NAME>
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View in Browser:**
   Open your browser and navigate to `http://localhost:5173` (or whichever port Vite assigns).
